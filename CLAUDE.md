# Claude Code 常設ルール（本プロジェクト専用）

## 0. 前提（最優先）
- 本ファイルは本プロジェクトにおける最優先ルールとする。
- 推測・一般論・改善案の羅列は禁止。事実（証跡）ベースのみ。
- 未指示のリファクタリング、最適化、命名変更、整形は禁止。
- ユーザーが明示していない UI / 機能 / 入力項目 / データの
  「削除・非表示・無効化」を行わない。
- 既存データ互換の破壊は禁止。
- 変更は原則 1テーマ・最大 2〜3 ファイルまで。

---

## 1. 調査フェーズ（READ ONLY）
### 目的
原因特定のみを行う。**編集禁止**。

### ルール
- ファイルの編集は禁止。
- 出力は以下の順で、簡潔に書く。
  1. 結論（1〜2行）
  2. 根拠（事実のみ）
  3. 該当箇所（ファイル名 : 行番号 + 関数名）
  4. 最短再現手順（最大5行）
  5. 次の最短修正方針（最大3行、推測なし）

### 禁止事項
- 「〜かもしれない」「一般的には」などの表現
- 改善案・別案の提示
- 長文説明

不明点がある場合は  
**「不足証跡：○○」のみを出力** する。

---
