WEB PORTFOLIO 制作 ( 2024.8- )

今回ポートフォリオを更新するにあたり、web制作に挑みました。

Webサービスなどのテンプレートは使わずclaudeを使用しコーディング。基礎から様々なJavaScriptも学びながら制作。

ウェブポートフォリオの特徴と今後の展望

1. 見て、触れて楽しいページ制作
  • 軽快な表示: Intersection Observer APIなどを活用し、画像表示を最適化。
  • 動きのある表現: Three.jsやGSAPを活用し、画像の滑らかな切り替えや変形を実現。
2. 表現力と技術力で、オリジナリティを追求
  • 動的デザイン：Canvas APIやCSS Maskを駆使し、背景や画像に動きを加えました。
  • レスポンシブデザイン：Flexbox、Grid、CSS変数を活用し、様々な画面サイズに対応。
3. 今後の課題と展望
  • 多様なデバイスでの最適な表示に継続して取り組みます。
  • 技術の進化に合わせ、新しい表現方法や最適化技術を導入していきます。
様々なクリエイティブ経験を活かし、ユーザー体験を重視したコンテンツ制作に取り組んでいきます。
